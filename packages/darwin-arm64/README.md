# @turndown-node/darwin-arm64

Native binding for macOS ARM64 (Apple Silicon).

This is a platform-specific package for [turndown-node](https://www.npmjs.com/package/turndown-node). You should not install this package directly.

## Installation

```bash
npm install turndown-node
```

The correct native binding will be installed automatically based on your platform.

## Documentation

See the main package: [turndown-node](https://github.com/sebastian-software/turndown-node)
