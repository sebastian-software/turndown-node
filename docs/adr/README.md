## ADR Format

Jeder ADR folgt diesem Format:

- **Status**: Proposed, Accepted, Deprecated, Superseded
- **Kontext**: Warum diese Entscheidung nötig war
- **Entscheidung**: Was wurde entschieden
- **Konsequenzen**: Positive und negative Auswirkungen
