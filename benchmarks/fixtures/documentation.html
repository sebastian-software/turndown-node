Installation

Install via npm:

npm install turndown-node

Or with yarn:

yarn add turndown-node

Or with pnpm:

pnpm add turndown-node

Quick Start

CommonJS

const TurndownService = require('turndown-node');

const turndownService = new TurndownService();
const markdown = turndownService.turndown('<h1>Hello World</h1>');
console.log(markdown);

ES Modules

import TurndownService from 'turndown-node';

const turndownService = new TurndownService();
const markdown = turndownService.turndown('<p>Hello <strong>World</strong></p>');

API

turndown(html)

Convert an HTML string to Markdown.

Parameter Type Description
html string The HTML string to convert

Returns: string - The converted Markdown

keep(filter)

Keep elements as HTML instead of converting them.

turndownService.keep(['del', 'ins']);
turndownService.turndown('<p>Hello <del>World</del></p>');
// => "Hello <del>World</del>"

remove(filter)

Remove elements entirely from the output.

turndownService.remove(['script', 'style']);
turndownService.turndown('<p>Hello</p><script>alert("!")</script>');
// => "Hello"

escape(text)

Escape Markdown special characters in a string.

turndownService.escape('*not emphasis*');
// => "\\*not emphasis\\*"

Options

Pass options to the constructor:

const turndownService = new TurndownService({
  headingStyle: 'atx',
  codeBlockStyle: 'fenced',
  bulletListMarker: '-',
  emDelimiter: '*',
  strongDelimiter: '**',
});
Option Type Default Description
headingStyle 'setext' | 'atx' 'setext' Style for headings
codeBlockStyle 'indented' | 'fenced' 'indented' Style for code blocks
bulletListMarker '*' | '-' | '+' '*' Marker for bullet lists
emDelimiter '_' | '*' '_' Delimiter for emphasis
strongDelimiter '**' | '__' '**' Delimiter for strong
