# turndown-core

Core Markdown AST and serialization library for the turndown ecosystem.

This crate provides the fundamental data structures and serialization logic used by other turndown crates.

## Features

- Markdown AST representation
- Configurable Markdown serialization
- Support for various Markdown elements (headings, lists, code blocks, etc.)

## Usage

This crate is primarily used as a dependency by other turndown crates. For end-user HTML-to-Markdown conversion, see the [turndown-node](https://www.npmjs.com/package/turndown-node) npm package.
